- made sure that for machine learning tasks, only significant features remained.
"""

"""### Dealing with missing Values
- Columns from both datasets that had more than 50% missing values were removed.
The missing values were filled in:

- Columns that are categorical and include their mode.
  - Columns of numbers with their average.


"""

# Drop columns with more than 50% missing values
threshold_two_years = 0.5 * len(compas_two_years_df)
threshold_two_years_violent = 0.5 * len(compas_two_years_violent_df)

for column in compas_two_years_df.columns:

        compas_two_years_df[column] = compas_two_years_df[column].fillna(compas_two_years_df[column].mode()[0])
    else:
        compas_two_years_df[column] = compas_two_years_df[column].fillna(compas_two_years_df[column].mean())

for column in compas_two_years_violent_df.columns:
    if compas_two_years_violent_df[column].dtype == 'object':
        compas_two_years_violent_df[column] = compas_two_years_violent_df[column].fillna(compas_two_years_violent_df[column].mode()[0])

- For `compas_two_years_df`, the following columns are encoded: `sex`, `age_cat`, `race`, etc.

- For `compas_two_years_violent_df`, the following columns are encoded: `sex`, `age_cat`, `race`, etc. encoded characteristics that have been concatenated back into the corresponding datasets.

- Use `StandardScaler` to standardise numerical features:
- All numeric columns were scaled for `compas_two_years_df`, with the exception of the target variable `two_year_recid`.

- All numeric columns were scaled for `compas_two_years_violent_df`, with the exception of the target variable `is_violent_recid`.

- Using `PolynomialFeatures`, second-degree interaction terms were created for a subset of numeric characteristics ({age`, `priors_count`, `decile_score`, etc.).

"""### Class Balancing - Applied undersampling to address class imbalance in target variables:
  - `two_year_recid` in `compas_two_years_df`.

  - `is_violent_recid` in `compas_two_years_violent_df`.
- Ensured both classes have equal representation by downsampling the majority class.


"""

# Class Balancing
def undersample_df(df, target_column):
    class_counts = df[target_column].value_counts()
    if len(class_counts) < 2:
        raise ValueError(f"The target column '{target_column}' does not have both classes (0 and 1).")
    minority_class_size = class_counts.min()
    majority_class = df[df[target_column] == class_counts.idxmax()]
    minority_class = df[df[target_column] == class_counts.idxmin()]
    majority_downsampled = resample(majority_class, replace=False, n_samples=minority_class_size, random_state=42)
    balanced_df = pd.concat([majority_downsampled, minority_class]).sample(frac=1, random_state=42).reset_index(drop=True)
    return balanced_df

compas_two_years_balanced_df = undersample_df(compas_two_years_df, 'two_year_recid')
compas_two_years_violent_balanced_df = undersample_df(compas_two_years_violent_df, 'is_violent_recid')

"""# 4. Model Development

### Data Preparation:
- After separating the features ({X`) and target ({y`), a 70-30 train-test split is performed.
- To guarantee interoperability with machine learning models, non-numeric columns are eliminated.

- The data is prepared for training once it was confirmed that every feature in `X_train` and `X_test` is numerical.
"""

# Data Preparation
X = compas_two_years_balanced_df.drop(columns=['two_year_recid'])  # Features
y = compas_two_years_balanced_df['two_year_recid']  # Target

# Split into Training and Testing Sets
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42, stratify=y)

# Ensure all non-numeric columns are encoded or dropped
non_numeric_columns = X_train.select_dtypes(include=['object', 'datetime']).columns

if len(non_numeric_columns) > 0:
    print(f"Non-numeric columns detected: {non_numeric_columns}")
    X_train = X_train.drop(columns=non_numeric_columns, errors='ignore')
    X_test = X_test.drop(columns=non_numeric_columns, errors='ignore')
else:
    print("No non-numeric columns detected in the training and testing datasets.")

# Verify all features are numeric
print(f"Remaining non-numeric columns in X_train: {X_train.select_dtypes(include=['object', 'datetime']).columns}")
print(f"Remaining non-numeric columns in X_test: {X_test.select_dtypes(include=['object', 'datetime']).columns}")

"""### Visualisation and Assessment of the Model
- Metrics including accuracy, precision, recall, F1 score, and ROC AUC score are calculated using the **Evaluation Function**. shows confusion matrices and classification reports.
- The Confusion Matrix Visualisation is a tool that A heatmap gives each model's prediction accuracy a clear, visual depiction.





"""

# Function to Evaluate Models
def evaluate_model(y_true, y_pred, model_name):
    print(f"\nEvaluation Metrics for {model_name}:")
    print(f"Accuracy: {accuracy_score(y_true, y_pred):.2f}")
    print(f"Precision: {precision_score(y_true, y_pred):.2f}")
    print(f"Recall: {recall_score(y_true, y_pred):.2f}")
    print(f"F1 Score: {f1_score(y_true, y_pred):.2f}")
    print(f"ROC AUC Score: {roc_auc_score(y_true, y_pred):.2f}")
    print("\nClassification Report:")
    print(classification_report(y_true, y_pred))
    print("\nConfusion Matrix:")
    print(confusion_matrix(y_true, y_pred))

# Plot confusion matrix heatmap
def plot_confusion_matrix(cm, model_name):
    plt.figure(figsize=(6, 5))
    sns.heatmap(cm, annot=True, fmt='d', cmap='Blues',
                xticklabels=['No Recidivism', 'Recidivism'],
                yticklabels=['No Recidivism', 'Recidivism'])
    plt.title(f'Confusion Matrix for {model_name}')
    plt.xlabel('Predicted')
    plt.ylabel('Actual')
    plt.show()

"""### Logistic Regression- **Training**:
-  The algorithm was trained using up to 1000 iterations.
- **Evaluation**: Measures such as recall, accuracy, precision, and ROC AUC are calculated for the test set.


- **Visualisation**: The confusion matrix heatmap provides a detailed study of the forecasts.



"""

# Logistic Regression
print("\nTraining Logistic Regression Model...")
log_reg_model = LogisticRegression(max_iter=1000, random_state=42)
log_reg_model.fit(X_train, y_train)
y_pred_log_reg = log_reg_model.predict(X_test)

evaluate_model(y_test, y_pred_log_reg, "Logistic Regression")

cm_log_reg = confusion_matrix(y_test, y_pred_log_reg)
# Plot for Logistic Regression
plot_confusion_matrix(cm_log_reg, "Logistic Regression")

"""### Decision Tree Classifier
- **Training**: A random state of 42 and default parameters were used to train the Decision Tree Classifier.
- **Evaluation**: For the test set, metrics such as ROC AUC, F1 score, recall, accuracy, and precision are calculated.

The classification performance is visualised using a confusion matrix heatmap.


"""

# Decision Tree
print("\nTraining Decision Tree Classifier...")
decision_tree_model = DecisionTreeClassifier(random_state=42)
decision_tree_model.fit(X_train, y_train)
y_pred_decision_tree = decision_tree_model.predict(X_test)

evaluate_model(y_test, y_pred_decision_tree, "Decision Tree Classifier")

cm_decision_tree = confusion_matrix(y_test, y_pred_decision_tree)
# Plot for Decision Tree Classifier
plot_confusion_matrix(cm_decision_tree, "Decision Tree Classifier")

"""### Random Forest Classifier
- **Training**: A random state of 42 and 100 estimators were used to train the Random Forest Classifier.
- **Evaluation**: For test data, metrics like ROC AUC, F1 score, recall, accuracy, and precision are computed.

Confusion matrix heatmaps offer valuable insights into categorisation outcomes.


"""

# Sanitize column names for compatibility with XGBoost
X_train.columns = X_train.columns.astype(str).str.replace('[\[\]<]', '', regex=True)
X_test.columns = X_test.columns.astype(str).str.replace('[\[\]<]', '', regex=True)

# Random Forest Model
print("\nTraining Random Forest Model...")
random_forest_model = RandomForestClassifier(n_estimators=100, random_state=42)
random_forest_model.fit(X_train, y_train)
y_pred_rf = random_forest_model.predict(X_test)

evaluate_model(y_test, y_pred_rf, "Random Forest")
cm_rf = confusion_matrix(y_test, y_pred_rf)

# Plot for Random Forest
plot_confusion_matrix(cm_rf, "Random Forest")

"""### XGBoost Classifier
- **Training**: Using `logloss` as the evaluation metric, the XGBoost Classifier was trained with default hyperparameters.
- **Evaluation**: Metrics that demonstrate the model's performance on the test set include accuracy, precision, recall, F1 score, and ROC AUC.

- **Visualisation**: The prediction results for recidivism classification are displayed in a confusion matrix heatmap.


"""

# XGBoost Model
print("\nTraining XGBoost Model...")
xgboost_model = XGBClassifier(use_label_encoder=False, eval_metric='logloss', random_state=42)
xgboost_model.fit(X_train, y_train)
y_pred_xgb = xgboost_model.predict(X_test)

evaluate_model(y_test, y_pred_xgb, "XGBoost")

cm_xgb = confusion_matrix(y_test, y_pred_xgb)

# Plot for XGBoost
plot_confusion_matrix(cm_xgb, "XGBoost")

# Preprocessing for Fairness Evaluation

def preprocess_aif360_data(X, y, sensitive_feature):
    print("\nHandling missing values and encoding data...")

    # Ensure X and y are DataFrames
    X = pd.DataFrame(X)
    y = pd.DataFrame(y, columns=["two_year_recid"])
    sensitive_feature = pd.DataFrame(sensitive_feature, columns=["race_original_x"])

    # Fill missing values
    X_numeric = X.select_dtypes(include=["number"]).fillna(X.mean())
    X_categorical = X.select_dtypes(exclude=["number"]).fillna("Unknown")

    # Encode categorical columns
    if not X_categorical.empty:
        encoder = OneHotEncoder(sparse=False, drop="first")  # Drop first to avoid multicollinearity
        encoded_categorical = encoder.fit_transform(X_categorical)
        encoded_categorical_df = pd.DataFrame(
            encoded_categorical, columns=encoder.get_feature_names_out(X_categorical.columns)
        )
        X = pd.concat([X_numeric.reset_index(drop=True), encoded_categorical_df.reset_index(drop=True)], axis=1)
    else:
        X = X_numeric

    # Encode sensitive feature
    sensitive_feature_encoded = LabelEncoder().fit_transform(sensitive_feature.values.ravel())
    sensitive_feature_encoded = pd.DataFrame(sensitive_feature_encoded, columns=["race_original_x"])

    # Combine all data into a single DataFrame
    combined_df = pd.concat([X.reset_index(drop=True), y.reset_index(drop=True), sensitive_feature_encoded.reset_index(drop=True)], axis=1)

    # Final check for any remaining NaN values and fill with appropriate defaults
    combined_df.fillna(0, inplace=True)  # Fill numerical NaNs with 0
    return combined_df

"""### AIF360 BinaryLabelDataset Preparation
1. **Goal**: Convert a cleaned DataFrame into a `BinaryLabelDataset` that is compatible with AIF360.
2. **Requirements** - Verify that the DataFrame has no missing values.
   Verify that numeric data types are present in every column.
3. **Favourable Label**: 1 (no recidivism, for example) in the 4. **BinaryLabelDataset**.
   - **Unfavourable Label**: 0 (recidivism, for example).
   Select the sensitive characteristic (e.g., race_original_x) in the **Protected Attribute** field.
4. **Validation**: Indicates errors if non-numeric or missing data is found.

"""

def prepare_aif360_dataset(df, label_column, protected_attribute):
    print("\nPreparing AIF360 BinaryLabelDataset...")

    # Check for missing values
    if df.isna().any().any():
        raise ValueError("DataFrame contains missing values. Please handle them before creating BinaryLabelDataset.")

    # Ensure all values are numeric
    if not np.issubdtype(df.dtypes.values.all(), np.number):
        raise ValueError("DataFrame contains non-numeric values.")

    # Create BinaryLabelDataset
    dataset = BinaryLabelDataset(
        favorable_label=1,
        unfavorable_label=0,
        df=df,
        label_names=[label_column],
        protected_attribute_names=[protected_attribute]
    )
    return dataset
